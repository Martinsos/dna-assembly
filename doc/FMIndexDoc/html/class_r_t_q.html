Detailed Description

Interface for RTQ geometric structure.

RTQ is geometric data structure that contains set of 2D-grid points Q and set of values (v, k) where each value (v,k) corresponds to one point from Q. Supports orthogonal range queries in Q and returns corresponding data (v,k) for each matched point.
