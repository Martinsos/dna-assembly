Detailed Description

StringView is slightly changed view of another string with minimum copying involved.

StringView can not affect or change viewed string. We can add suffix, preffix, or limit our view to substring of viewed string. When making a view of another string, no copying is done. When adding prefix or suffix they are actually stored, so prefix and suffix should not be big.
