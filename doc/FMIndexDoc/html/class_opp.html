vector< OppRows > Opp::findRowsForSuffixes ( const StringView &  P) const

Same as method findRows() but returns rows for all suffixes of P, not just P.

Applies findRows() for each suffix of given string.

IMPORTANT: same time complexity as oppRows (O(p+occ), not |P| * O(oppRows) as you would expect).

Returns:
vector[m] corresponds to P[p-m,p]
vector< OppRows > Opp::findRowsForSuffixesWithPrefix ( const StringView &  P,
char  C 
) const

Same as method findRows() but returns rows for all suffixes of P prefixed with character C.

Like findRowsForSuffixes(), but adds prefix to each suffix before searching.

Example: C = '$', P = "abcd", returns rows for: $d, $cd, $bcd, $abcd IMPORTANT: same time complexity as oppRows (O(p+occ), not |P| * O(oppRows) as you would expect).

Returns:
vector[m] corresponds to C+P[p-m,p]
