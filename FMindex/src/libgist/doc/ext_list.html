libgist v.1.0 Extensions

At this point, only the B-tree and R-tree extensions, which are part of the standard libgist distribution, are available. We will add new extensions to this list as they become available. If you have developed a GiST extension yourself that you think might be useful for others, please send us email, so we can post it on this page.

List of extensions

  • R-trees
